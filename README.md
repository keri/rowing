# rowing
A rowing app to track and make predictions for optimal rower and race performance

##Tracks Data
###Race

Time
Temp
Wind (high,med,low)
Date
Location
Type of row (scull/sweep)
Number of people in boat
Boat Name
Rower Names
Regatta Name
Number of Rowers in Boat
Coxswain Name
Level of Race (Novice/Inter)
Age Group
First (1st/2nd/3rd/4th/5th/6th/7th, etc)
Semi (1st/2nd/3rd/4th...)
Final A,B, or C decided from above (1st/2nd/3rd...etc)

###Rower
Erg 2K
Erg Split
Erg Power
Missed morning practices (#)
Missed afternoon practices (#)
Weight
Age
Personality (Only asks questions, doesn't interpret results, records answers for model)
Date
Level (Novice, Int, Senior)
